# TKCG

一个文本内容生成器，使用Java8开发，
配置文件使用YAML格式，模板文件使用FreeMarker格式
通过数据组件和个性化配置，可以用在不同场景中。
生成项目骨架（前端，后端，移动，中间件等，不限语言框架）
生成业务代码（比如Java程序常见的Model,DAO,DTO,Service等）
生成测试代码
生成接口文档
生成数据库文档
